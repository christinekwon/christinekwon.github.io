The Fibonacci garden is a generative garden of spirals created using the Fibonacci sequence.
At the heart of this project is an exploration of 3D representation methods of fractals. I became captivated by fractal patterns because of their ability to create extremely complex looking results with very simple rules. I desired to experiment with the Fibonacci sequence for this project because I am particularly interested in the beautiful visual effects it creates in nature, as can be seen through flower petal / seed arrangement, shells, and more.
Each time the website is loaded, 20 unique spirals are generated.
Each spiral is created by first creating a stacked cube structure based on the fibonacci sequence..
.. and then using the corners of the cubes to draw BÃ©zier curves.
The user may pan around the site to observe the garden from various angles.
